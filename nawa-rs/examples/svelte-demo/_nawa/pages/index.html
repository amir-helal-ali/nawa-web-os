كيف يعمل الدمج؟

  1. Build-time: SvelteKit يُجمَّع عبر adapter-nawa
  2. الناتج: _nawa/ directory يحوي manifest + HTML + JS/CSS
  3. Runtime: nawad يقرأ manifest ويُخدم الصفحات
  4. Hydration: المتصفح يُفعّل SvelteKit client-side
  5. Live updates: WebSocket يدفع التحديثات فوراً
